# app
## Integrated application with server, audio, vision, and gui components  
To run the app in console mode execute the following commands:  
* `python engine.py`
* `python audio.py`  
* `python console.py`  
Try some commands in the console  
`audio a`  
`engine select=1`  
`audio command=quit`  
`engine command=quit`  
`quit`
